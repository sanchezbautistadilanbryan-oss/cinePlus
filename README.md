# cinePlus
Aplicacion basica de cobro para un cine creada en python con wxglade.Consta de 4 ventanas:inicio de secion,taquilla,dulceria y ticket 

ventana de inicio de Sesion (login)
--El sistema de muestra una pantalla para iniciar sesion
--Para este proyecto,el inicio de sesion se hara con datos personalizados (como nombre:usuario y contraseña:131)

Ventana de Boletos (Taquilla)
--El sistema debe permitir que el empleado seleccione la pelicula,el horario y sala
"F1" 11:00,13:00,15:00 sala 1
"Karate Kid" 12:00,14:00,17:00 Sala 2
"Cars" 10:30,12:00,14:00 sala 3

--se tiene que especificar la cantidad de boletos.
--se debe especificar si los boletos son para niños o adultos ya que tienen un precio diferente(adultos:80,menores:70)
--Al terminar la sesion de boletos el sistema debe mostrar un subtotal de los boletos adquiridos.

ventana de productos(Dulceria)
--El sistema te muestra una pantalla donde te permite selccionar los combos y productos que desees.
Refrescos:chicos,medianos y grandes piña,naranja  fresa
combos,palomitas grandes y medianas

ventana final(Ticket)
Te muestra el subtotal y total de tus boletos y productos finales asi como la sala,pelicula y el horario
